# ts学习

# 1. ts与js的对比

# TypeScript是一个应用程序级的JavaScript开发语言。（更适合开发大型应用）

# TypeScript是JavaScript的超集，可以编译成纯JavaScript。

# TypeScript跨浏览器、跨操作系统、跨主机、且开源。

# TypeScript遵循JavaScript的语法和语义，所以对于我们前端来说，学习起来并没有太大的难度。

# TypeScript可以重用JavaScript代码，调用流行的JavaScript库。

# TypeScript提供了类、模块和接口，更易于构建组件和维护。

# 2. 开发环境安装

  1. 安装node
  2. 安装typescript包
npm install typescript -g
# 查看ts版本
tsc --version

# 3. helloWorld

  1. 初始化项目
npm init -y
  1. 根目录创建配置文件，可以通过读取它来设置TypeScript编译器的编译参数,与.babelrc和.eslintrc差不多
# 创建配置文件
tsc --init
  1. 安装@types/node,这个主要是解决模块的声明文件问题
npm install @types/node --D
  1. 创建helloWorld.ts
var helloWorld:string = "helloWorld"
console.log(helloWorld)
  1. 使用vscode中任务/运行任务,然后选择tsc：构建-tsconfig.json，生成helloWorld.js
  2. 在终端中输入node helloWorld.js即可
