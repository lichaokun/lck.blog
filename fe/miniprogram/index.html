# 小程序入门

# 1. 小程序的特点

特点

  1. 小程序适合做简单，用完即走的应用
  2. 小程序适合低频的应用
  3. 小程序适合性能要求不高的应用

注意点

  1. 只有text标签包裹了的才能在手机中长按选中复制
  2. 每个页面最外层包裹了一层page标签
  3. rpx单位和rem差不多，最好是以750宽设计稿来定，字体可以不用rpx，根据情况来定
